# Web Scaffold Notes

This folder is included as an optional Flutter web scaffold.

## Scope

- Basic `index.html` and `manifest.json`
- No production web hosting/config is assumed
- No personal backend keys or Firebase project IDs are embedded here

## If you want web support

1. Run and test with `flutter run -d chrome`
2. Add/verify web icons and PWA assets
3. Configure hosting, base href, and caching strategy
4. If enabling Firebase, provide your own web Firebase configuration
