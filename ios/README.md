# iOS Scaffold Notes

This folder is included as an optional Flutter iOS scaffold.

## What is included

- Standard Flutter iOS Runner project structure
- Placeholder bundle identifiers (`com.example.stridemind`)
- No personal Firebase project binding

## Before releasing iOS

1. Set your own Apple Team and signing profile in Xcode
2. Set your own bundle identifier
3. If using Firebase, replace iOS Firebase config for your project
4. Build and test on a real iOS device / TestFlight

Generated local files under `ios/Flutter` are intentionally git-ignored.
